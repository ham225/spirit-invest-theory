## 毎日のデータ収集+バックテストの仕組み

平日 日本時間18:30 に GitHub Actions が自動で動き、市場データと理論シグナル(Layer1/2/3)を
`data/log.csv` に1日1行ずつ記録していく。LLMは使わず、無料のデータ取得(yfinance)と
決定的な計算のみなので運用コストは¥0。

| ファイル | 役割 |
|---|---|
| `THEORY.md` | **理論の正式文書(v1.0)**。仮説・スコア定義・統合ロジック・年間シナリオ生成手順・反証条件 |
| `theory_signals.py` | 干支・五行・九星気学(Layer1/2)を計算式で自動算出。西洋占星術(Layer3)は年ごとの暦データを参照 |
| `theory_score.py` | THEORY.mdの計算式の実装。日次警戒レベル(`python theory_score.py 2026-11-01`)と年間シナリオ(`python theory_score.py --year 2026`)を出力 |
| `astro_events_<year>.json` | その年の水星逆行期間・主要トランジット日程(手入力。**新しい年になったら追加が必要**) |
| `fetch_data.py` | 日経平均・東京エレクトロン(火=AI半導体の代表)・ドル円(水)・金(資源)の値動き+その日の理論シグナルを取得し `data/log.csv` に追記 |
| `backtest.py` | `data/log.csv` を集計し、水星逆行中/通常時・土星海王星の合の前後/通常時で値動きの大きさを比較した `insights/backtest.md` を再生成(データが20日分たまるまではプレースホルダー) |
| `.github/workflows/daily.yml` | 平日18:30(JST)に上記2スクリプトを自動実行し、結果をリポジトリにコミット・プッシュ |
| `data/log.csv` | 日次ログ(自動・追記のみ) |
| `insights/backtest.md` | 集計レポート(自動・毎回上書き) |

### 使い方
- 今日のデータ・理論との照合結果を見たい: `insights/backtest.md` を開く(データが貯まるほど中身が充実する、株初動ピックアップの「法則性.md」と同じ考え方)
- 手動で今すぐ実行したい: GitHubリポジトリの Actions タブ → 「毎日データ収集+バックテスト更新」→ Run workflow
- 来年以降も使う場合: `astro_events_2027.json` のように新しい年のトランジット暦を追加すること(自動計算できるのはLayer1・Layer2まで)

## 三層スピリチュアル・マーケット理論(概要。正式な定義は [THEORY.md](THEORY.md))

森永氏のSMCとの違い: SMCは属人的なキャラクターコンテンツだが、こちらは**誰でも毎年同じロジックで機械的に年間シナリオを再現できるフレームワーク**として設計。将来の自動記事化や自分の投資参考メモへの転用を想定。

### Layer 1|干支五行トレンドレイヤー(方向・強度、年次解像度)
その年の十干十二支の五行属性(木火土金水)を特定し、属性が重なる年(比和)はそのエネルギーに紐づく資産クラスのトレンドが増幅されやすい、という仮説。

- 火=AI・半導体・グロース株などモメンタム系 / 水=金融・流動性・為替 / 金=貴金属・資源 / 木=新興国・成長 / 土=不動産・インフラ
- 十二支相場格言(「辰巳天井、午尻下がり」等)をクロスチェック層として併用
- **2026年=丙午(火の比和)** → 「値動きが一方向に激しくなりやすい年」という仮説

### Layer 2|九星気学・中宮レイヤー(隠れリスク層、年次解像度)
その年の中宮星(年盤)の五行と、干支の五行の相生相剋関係を見る。相剋関係だと、表面的トレンドに対する反動・急変・伏在リスクの顕在化確率が上がるという「地雷リスク指数」を出す(方向は決めない、警戒フラグのみ)。

- **2026年=中宮一白水星(水)が丙午(火)を剋す** → 「前半形成されたトレンドの水面下に、後半噴出しうるリスク要因がある」という仮説
- **この層は既存の先行事例が見つからず、理論の独自性の核**(Layer1は複数のアナリストが既に扱っているが、Layer2の切り口はオリジナル)

### Layer 3|西洋占星術トランジットレイヤー(タイミング層、月次解像度)
アウタープラネットの大きなアスペクト(土星×海王星など)を長期の潮目転換候補時期、水星逆行期間を短期的な「見誤り・訂正が起きやすい」警戒ウィンドウとして使う。

- 土星海王星の合=2026年2月20日exact(前半)
- 水星逆行は2026年3回のうち後半に2回集中(6/30〜7/24 蟹座、10/24〜11/13 蠍座)

### 統合ロジック
Layer1(方向・強度)× Layer2(反転含みの警戒フラグ)× Layer3(いつ顕在化しやすいか)を掛け合わせ、「トレンド継続 or 反転」の予言ではなく **「いつリスク管理を強化すべきか」という運用シナリオ** に変換するのがこの理論の核。

### 2026年後半への適用例(仮説であり投資助言ではない)
前半に一方向へ伸びたテーマ(AI・半導体等の火属性資産)は、後半、特に水星逆行が重なる**7月前後・11月前後**に、隠れていたリスク材料の顕在化をきっかけに振れ幅が拡大しやすい。トレンド継続・逆張りいずれの立場でも、この2つの時期はポジションサイズを抑えリスク管理を強化する、という運用ルールが導ける。

## 事実関係の検証結果

| 項目 | 検証結果 | 信頼度 |
|---|---|---|
| 2026年の中宮=一白水星 | 9年周期のルールと整合(2008・2017年も同じく一白水星年)。複数サイトで一致 | 中〜高 |
| 丙午=火の比和 | 丙(陽の火)・午(陽の火)とも五行で火に分類。丙午迷信は1966年の出生率減少と関連 | 高 |
| 「辰巳天井、午尻下がり」 | 実在する十二支相場格言(みずほ証券・日経で解説)。統計的裏付けは限定的との注記あり | 高 |
| 「水火激突」という言葉 | 正式な九星気学用語としては未確認。近い概念に「水火殺」(一白水星×九紫火星の凶方位)があり、「水火激突」は森永氏の造語的表現の可能性が高い | 中 |
| 2026年水星逆行3回の日程 | 2/26-3/20(魚座)、6/29or30-7/23or24(蟹座)、10/24-11/13(蠍座)。海外サイトと日本語占星術サイトの差は時差のみ | 高 |
| 土星海王星の合(2026/2/20 exact) | 会合周期約36年は事実。牡羊座0度での合という希少性の年代には情報源間でばらつきあり(誇張の可能性) | 中〜高 |
